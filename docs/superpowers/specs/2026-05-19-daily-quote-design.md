# 每日一句 — 设计规格

## 概述

一个纯单文件 HTML 页面。打开后展示一句当日名言（选自哲学/心理学/文学经典），附带出处，下方提供复制按钮。每天自动切换，同一日期始终展示同一句话。

## 技术方案

单文件 HTML，零外部依赖。所有内容（名言库、样式、逻辑）内联在一个 `.html` 文件中，双击浏览器即可打开。

## 名言库

- 规模: 60–90 条
- 语言: 中文为主，含西方经典译文
- 来源覆盖: 中国哲学（道德经、庄子、论语、禅宗语录）、西方哲学（尼采、加缪、斯多葛学派）、心理学（荣格、阿德勒、弗洛姆、弗兰克尔）
- 每条结构: `{ text: "名言正文", source: "出处/作者" }`
- 例句: `{ text: "认识你自己。", source: "古希腊德尔斐神庙铭文" }`

## 日期映射

- 取当前日期 `YYYY-MM-DD` 字符串
- 用 DJBA2 哈希算法将日期转为整数，对名言数组长度取模
- 确保: 同一日期 → 同一索引；不同日期均匀分布

## 视觉设计

- 背景: 暖色奶油/米白，模仿旧书页质感
- 文字: 深灰偏黑 (#2c2c2c)
- 名言字体: Georgia / 思源宋体（衬线），字号 clamp(1.5rem, 3vw, 2.5rem)
- 出处字体: 无衬线，小一号，颜色略淡，以 em-dash（——）前缀
- 布局: flexbox 水平垂直居中，大量留白
- 复制按钮: 右下角固定定位，小尺寸、低不透明度，hover 时显现；复制后短暂反馈"已复制"

## 响应式

- 桌面: 居中卡片，最大宽度约 680px
- 移动: 自适应宽度，保持留白比例
- 极端窄屏 (< 320px): 最小字号限制，保证可读

## 交互

- 复制按钮: 点击后调用 `navigator.clipboard.writeText()` 复制"名言 + 出处"；不支持时回退 `document.execCommand('copy')`
- 无其他交互元素

## 文件结构

```
每日一句/
└── index.html   (元数据 → CSS → HTML结构 → JS)
```

## 边界处理

| 场景 | 处理方式 |
|------|----------|
| 哈希值超出数组范围 | 取模运算，数学保证不越界 |
| Clipboard API 不可用 | 回退到 execCommand('copy') |
| JS 被禁用 | `<noscript>` 提示用户启用 JS |
| 极端窄屏 (< 320px) | min-width / min-font-size 保护 |
